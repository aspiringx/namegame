# NameGame

**The relationship game that starts with a name.**

NameGame makes it fun for people in a group to meet, remember names, and get acquainted.

See home page at [namegame.app](https://namegame.app).

If you're in software or want to be (software engineeer, UX, product management, etc.)
NameGame is an open source AspiringX project you can be part of to learn, 
contribute, etc.
